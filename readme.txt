# 🏋️‍♂️ Gym AI Chatbot Assistant

An AI-powered assistant that allows gym managers to query their member, payment, and cancellation data in natural language. The bot dynamically detects user intent using `spaCy` and returns intelligent responses — without hardcoding each question.

---

## 🚀 Features

- Natural language understanding (e.g., "Who joined last month?", "What was the revenue yesterday?")
- Conversational chatbot UI using Streamlit
- Intent classification using spaCy
- Intelligent CSV-based querying
- Tracks chat history
- Works fully offline
